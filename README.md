# 🎨 ColorVisionAI

An AI-powered computer vision application that detects colors and objects in real time using Python and OpenCV.

## Features

- Live webcam feed
- Real-time color recognition
- HSV color detection
- RGB and HSV values
- Confidence score
- Object detection
- Bounding boxes
- Modular architecture

## Technologies

- Python
- OpenCV
- NumPy

## Installation

```bash
pip install -r requirements.txt
python main.py
